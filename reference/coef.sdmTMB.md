# Get fixed-effect coefficients

Get fixed-effect coefficients

## Usage

``` r
# S3 method for class 'sdmTMB'
coef(object, complete = FALSE, model = 1, ...)
```

## Arguments

- object:

  The fitted sdmTMB model object

- complete:

  Currently ignored

- model:

  Linear predictor for delta models. Defaults to the first linear
  predictor.

- ...:

  Currently ignored
